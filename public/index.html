What is it?

davidhong is a fast and large JavaScript library, framework, compiler, everything that simplifies client-side scripting. It tirelessly enhances your JavaScript code using techniques that matter the most. It is a product of all existing, awesome libraries and frameworks – and is of course compatible with <insert your library/framework here>.

The best thing about it is that it only weighs: 160 pounds.

How does it work?

davidhong works by autonomously keeping up-to-date with latest and greatest technologies that allow anyone to take advantage of great new features of modern browsers. To name a few, ECMAScript 5 (and upcoming ES.next), client-side MVC with frameworks such as Backbone and Ember.js.

Diligently integrating with variety of server-side technologies is a huge factor in davidhong. We've had previous experiences with frameworks such as Spring, ASP.net MVC 3, NancyFx and node.js to name a few. We believe that figuring out vital integral points between client and server is just as important as the code itself.
